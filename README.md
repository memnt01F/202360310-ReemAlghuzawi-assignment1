# 202360310-ReemAlghuzawi-assignment1
Personal portfolio website (Assignment 1)


## Overview
This project is a responsive personal portfolio website built for Web Engineering Assignment 1. It presents personal information, selected projects, and a contact form in a clean and structured layout.

The website demonstrates foundational front-end development skills using HTML, CSS, and JavaScript.

---

## Technologies Used

- HTML5 – Structure and content
- CSS3 – Styling, layout, and responsive design
- JavaScript (ES6) – Theme toggle and form handling
- CSS Grid & Flexbox – Layout structure
- LocalStorage – Theme preference persistence

---

## Features

- Responsive layout for desktop and mobile
- Dark/Light theme toggle with persistent storage
- Project showcase section
- Contact form with JavaScript validation (demo submission message)
- Clean UI with modern styling

---

## How to Run

1. Download or clone the repository.
2. Open the `assignment-1` folder.
3. Open `index.html` in a web browser.
4. No additional setup or installation is required.
